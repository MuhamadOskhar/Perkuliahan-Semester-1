Algoritma "konversi detik"
{Algoritma ini digunakan untuk mengkonversikan detik ke dalam hari, jam, menit, dan detik}

DEKLARASI
detik : Integer
menit : Integer
jam   : Integer
hari  : Integer
sisaDetik   : Integer

ALGORITMA

- read(detik)

- hari = detik / 86400
- sisaDetik  = detik % 86400
- jam  = sisaDetik / 3600
- sisaDetik = sisaDetik % 3600
- menit = sisaDetik / 60
- detik = sisaDetik % 60

- write(hari, jam, menit, detik)
